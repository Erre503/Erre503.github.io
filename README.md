# Erre503 - Terminal GitHub Profile

Benvenuto nel mio terminale interattivo GitHub!

## 🖥️ Terminale Interattivo

Questa pagina presenta un'interfaccia terminale completamente nera che simula un ambiente Linux. Puoi interagire con il terminale usando comandi reali come in un vero terminale Unix.

## 💻 Comandi Disponibili

- `ls` - Lista i file nella directory corrente
- `cat <file>` - Mostra il contenuto di un file
- `pwd` - Mostra la directory corrente
- `whoami` - Mostra il nome utente corrente
- `cd <dir>` - Cambia directory
- `echo <text>` - Mostra testo
- `date` - Mostra la data corrente
- `uname` - Mostra informazioni sul sistema
- `neofetch` - Mostra informazioni dettagliate sul sistema
- `clear` - Pulisce il terminale
- `help` - Mostra tutti i comandi disponibili

## 📊 File delle Statistiche GitHub

- `github_stats.txt` - Statistiche di base del profilo GitHub
- `github_detailed.txt` - Statistiche dettagliate e analytics

### Come visualizzare le statistiche:

```bash
cat github_stats.txt
cat github_detailed.txt
```

## 🚀 Come Usare

1. Clicca sull'area del terminale per iniziare a scrivere
2. Digita un comando e premi Invio
3. Usa le frecce su/giù per navigare nella cronologia dei comandi
4. Usa `help` per vedere tutti i comandi disponibili

## 🛠️ Tecnologie Utilizzate

- **HTML5** - Struttura della pagina
- **CSS3** - Styling con tema terminale nero
- **TypeScript/JavaScript** - Logica del simulatore di terminale
